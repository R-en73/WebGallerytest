# 画像フォルダ

このフォルダには作品のサムネイル画像を配置してください。

## 推奨仕様

- **形式**: JPG, PNG, WebP
- **サイズ**: 400x250px 程度
- **ファイル名**: work1.jpg, work2.jpg, ... (main.jsのデータと対応)

## ファイル名の例

```
work1.jpg - コーポレートサイト制作
work2.jpg - ECサイトリニューアル
work3.jpg - 事業計画プレゼン資料
work4.jpg - ランディングページ制作
work5.jpg - 研修資料デザイン
work6.jpg - ブランドロゴデザイン
```

※ 画像がない場合は、プレースホルダーが自動的に表示されます。
