## 1. プロジェクト概要

### 1.1 目的
自身の制作物を一覧化し、視覚的に訴求するオンラインポートフォリオを構築する。
サイバーパンク風のダークテーマで作品のクリエイティブな印象を与える。

### 1.2 ターゲットユーザー
| ユーザー層 | 想定ニーズ |
|-----------|-----------|
| 採用担当者 | スキル・制作物の確認 |
| クライアント | 依頼可能な制作物の把握 |
| 技術コミュニティ | 技術力・アウトプットの参照 |

### 1.3 使用技術
- **HTML5**: セマンティックなマークアップ
- **CSS3**: ダークテーマ、ネオンエフェクト、レスポンシブデザイン、高度なアニメーション、スクロールアニメーション
- **JavaScript (Vanilla JS)**: DOM操作、フィルタ機能、モーダル表示、動画再生、Intersection Observer API

## 2. 画面構成（サイトマップ）

```
トップページ（ギャラリー）
 ├ ヘッダー（固定）
 │   ├ ロゴ / サイト名（グラデーション）
 │   └ ナビゲーション（Works, About, X）
 ├ ヒーローセクション
 │   ├ 背景アニメーション（グラデーション＋グリッド）
 │   ├ タイトル（大型タイポグラフィ）
 │   └ CTAボタン
 ├ カテゴリフィルタ（スティッキー）
 ├ 作品カードギャラリー
 │   └ 各作品カード
 │       ├ サムネイル画像/動画サムネイル
 │       ├ 再生アイコン（動画の場合）
 │       ├ 作品名
 │       └ カテゴリラベル
 ├ 作品詳細モーダル
 │   ├ 画像表示 or 動画プレイヤー
 │   ├ 作品タイトル
 │   ├ 詳細説明
 │   ├ PDFリンク（該当する場合）
 │   └ 外部サイトリンク（該当する場合）
 ├ Aboutセクション
 │   ├ 自己紹介
 │   └ 実績（プロジェクト数等）
 └ フッター
     ├ X（Twitter）リンク
     ├ ナビゲーションリンク
     └ Copyright
```

## 3. 機能要件

### 3.1 カード型ギャラリー
| 項目 | 詳細 |
|------|------|
| 表示形式 | 3列グリッドレイアウト（レスポンシブ対応） |
| カード要素 | サムネイル画像、作品名、カテゴリ、再生アイコン（動画用） |
| ホバーエフェクト | 浮き上がり、スケール拡大、グラデーションボーダー、ネオングロー |

### 3.2 カテゴリフィルタ
| カテゴリ | 説明 |
|---------|------|
| All | 全作品を表示 |
| AI動画 | AI生成動画作品 |
| Web | Web制作物 |
| PDF | ドキュメント系 |
| Other | その他の制作物 |

**動作仕様**:
- フィルタボタンをクリックで即座に絞り込み
- アクティブなフィルタはグラデーション背景で強調
- フェードイン/スケールアニメーション付き

### 3.3 検索機能
| 項目 | 詳細 |
|------|------|
| 検索対象 | 作品タイトル、説明文、カテゴリラベル |
| 検索方式 | リアルタイムインクリメンタル検索 |
| UI要素 | 検索ボックス（ネオンフォーカスエフェクト）、クリアボタン |
| キーボード操作 | Enter: フォーカス解除、Escape: 検索クリア |

**動作仕様**:
- フィルタと検索は併用可能（AND条件）
- 該当なしの場合は空メッセージを表示
- 検索クエリ入力でクリアボタンが表示

### 3.4 動画再生機能
| 項目 | 詳細 |
|------|------|
| サムネイル | 静止画サムネイル＋再生アイコンオーバーレイ |
| 再生方法 | モーダル内でHTML5 video要素を使用 |
| 操作 | 自動再生、コントロール表示、ループ再生 |

### 3.5 ライトボックス表示
| 項目 | 詳細 |
|------|------|
| トリガー | カードのクリック |
| 表示内容 | 画像または動画、詳細情報、各種リンク |
| 閉じ方法 | ボタン（回転アニメーション）、背景クリック、Escキー |

### 3.6 SNSリンク・X対応
| 項目 | 詳細 |
|------|------|
| 対象 | X（Twitter）アカウント |
| 配置場所 | ヘッダーナビゲーション、フッター |
| 動作 | `target="_blank"` で別タブ表示 |
| アイコン | Xロゴ（SVG）をグラデーション/ネオン表示 |

### 3.7 スクロールアニメーション
| 項目 | 詳細 |
|------|------|
| 実装方式 | Intersection Observer API |
| アニメーション種類 | fade-up, fade-down, fade-left, fade-right, scale-up, scale-down, rotate-in, blur-in |
| 遅延オプション | data-scroll-delay属性（1-6）による段階的表示 |
| スタガー効果 | ギャラリーカードの順次表示 |
| アクセシビリティ | prefers-reduced-motion対応 |

**対応セクション**:
- フィルターセクション（検索ボックス、フィルターボタン）
- Aboutセクション（タイトル、テキスト、統計）
- フッター（CTA、スキルバッジ、メインコンテンツ）
- ギャラリーカード（stagger animation）

### 3.8 レスポンシブ対応
| デバイス | 画面幅 | カラム数 |
|---------|--------|---------|
| スマートフォン | 〜767px | 1列 |
| タブレット | 768px〜1023px | 2列 |
| PC | 1024px〜 | 3列 |

**スマートフォン版フッター仕様**:
- 横並びコンパクトレイアウト採用
- フッターメインエリア: ブランドロゴ＋ナビゲーショングリッド横並び配置
- フォントサイズ・パディング縮小による省スペース化
- CTAセクション・スキルバッジ縮小表示

## 4. 非機能要件

### 4.1 パフォーマンス
| 項目 | 要件 |
|------|------|
| 画像最適化 | PNG/WebP形式、適切な圧縮 |
| 動画 | MP4形式、適切なビットレート |
| 遅延読み込み | `loading="lazy"` 属性を使用 |
| 初期表示 | 3秒以内を目標 |
| アニメーション | GPU最適化（will-change使用） |

### 4.2 保守性
| 項目 | 要件 |
|------|------|
| データ構造 | JavaScript配列での作品データ管理 |
| コード構成 | HTML/CSS/JSの分離 |
| CSS変数 | カラー・スペーシング等を変数化 |

### 4.3 アクセシビリティ
| 項目 | 要件 |
|------|------|
| 代替テキスト | 全画像にalt属性を付与 |
| キーボード操作 | Tabキーでの移動対応 |
| ARIAラベル | モーダル、ボタン等に適切なaria属性 |
| モーション設定 | prefers-reduced-motion尊重 |

## 5. デザイン・UI要件

### 5.1 デザインコンセプト
- **サイバーパンク**: ネオンカラーとダークテーマの組み合わせ
- **未来的**: グリッドパターン、スキャンライン、グリッチエフェクト
- **インパクト**: 大胆なタイポグラフィとグロー効果

### 5.2 カラーパレット
| 用途 | カラーコード | 説明 |
|------|-------------|------|
| 背景（Primary） | #030305 | メイン背景（漆黒） |
| 背景（Secondary） | #080810 | セクション背景 |
| テキスト（Primary） | #FFFFFF | メインテキスト |
| テキスト（Secondary） | #8B9CB3 | サブテキスト |
| アクセント（Cyan） | #00FFFF | メインアクセント、ネオン |
| アクセント（Purple） | #8B5CF6 | グラデーション中間色 |
| アクセント（Pink） | #FF00AA | グラデーション終点 |

### 5.3 グラデーション
```css
--accent-gradient: linear-gradient(135deg, #00ffff 0%, #8b5cf6 50%, #ff00aa 100%);
```

### 5.4 角丸の指針
| 要素 | 角丸サイズ |
|------|-----------|
| カード | 20px |
| ボタン | 50px（ピル型） |
| モーダル | 24px |
| タグ/ラベル | 20px |

### 5.5 エフェクト・アニメーション
| 要素 | アニメーション |
|------|---------------|
| カードホバー | 浮き上がり(-16px) + スケール(1.02) + ネオングロー |
| グラデーションボーダー | ホバー時に出現 |
| CTAボタン | シマーエフェクト + グロー拡大 |
| 背景 | グラデーションの微動 + hue-rotate |
| グリッドパターン | パルスアニメーション |
| モーダル閉じるボタン | 90度回転 |
| スクロールインジケーター | 上下アニメーション |
| スクロールアニメーション | fade-up/down/left/right, scale, rotate, blur |

## 6. データ構造

### 6.1 作品データスキーマ
```javascript
{
  id: Number,           // 一意のID
  title: String,        // 作品名
  category: String,     // カテゴリ（video/website/pdf/other）
  categoryLabel: String,// 表示用カテゴリ名
  thumbnail: String,    // サムネイル画像パス
  image: String,        // 詳細画像パス
  video: String,        // 動画ファイルパス（オプション）
  description: String,  // 作品説明
  url: String,          // 外部サイトURL（オプション）
  pdf: String,          // PDFファイルパス（オプション）
  date: String          // 制作日
}
```
